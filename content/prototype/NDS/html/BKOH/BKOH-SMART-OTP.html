    3. 3) 농협은행 선택 후 확인 버튼 선택
  • 스마트 OTP 공동앱 실행방법
    1. 1) 스마트 OTP 공동앱 실행 버튼 선택
    2. 2) 금융결제원 ‘스마트 OTP 공동앱’ 자동 실행
    3. 3) 스마트 OTP 카드를 스마트폰 뒷면에 접촉
    4. 4) 스마트 OTP 공동앱에서 생성된 OTP 인증번호를 입력
  • 스마트 OTP 공동앱 수동입력인 경우

    2. 2) 농협은행 선택
    3. 3) 일회용 PIN번호 입력
    4. 4) 스마트 OTP카드를 스마트폰 뒷면에 접촉
